

## 什么是StableVerse?

StableVerse is a locally run graph tool developed in C++. Suitable for beginners to quickly get started; Suitable for small-scale computing power scenarios of text and image production for office workers.

StableVerse是一个基于c++开发的本地运行的生图工具。适合于新手快速上手；适合于办公人群小规模算力下的文生图、图生图场景。



## Characteristics:

```shell
Novice friendly: Pre configured with multiple mainstream models to help beginners quickly enter their roles;
Selected cases: Integrating best practices to quickly generate images;
High performance: written in C++, with direct access to hardware;
Lightweight: smaller volume;
Offline: Independent model, customizable:
Privacy protection: Local model, run locally.
It can be used without restrictions or internet connection.
```



## 特点：

```shell
  新手友好：预置多个主流模型的配置，帮助新手尽快进入角色；
  精选案例：集成最佳实践，快速出图；
  高性能：c++编写，直接访问硬件；
  轻量级： 更小的体积；
  离线：独立模型，可任意定制：
  隐私保护：本地模型，本地运行。
  无限制、没有网络也可以用。
```



```shell
Platform support: Supports cross platform operation. Currently tested well under Windows. Theoretically supports Linux and Mac environments.
Hardware dependency: The hardware requires 6G of graphics memory from Nvidia graphics cards. Currently compatible with CUDA12.9

平台支持：支持跨平台运行。目前在windows下测试良好。理论上支持linux，mac环境。 
硬件依赖：硬件需要英伟达显卡6G显存。目前适配cuda12.9.
```
